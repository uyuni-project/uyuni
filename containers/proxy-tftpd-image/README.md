TFTP server for use with the Uyuni proxy or server.
This server is serving content from an HTTP API rather than from a local storage.

This image cannot be used independently.
Refer to the Uyuni proxy installation documentation either on [podman](https://www.uyuni-project.org/uyuni-docs/en/uyuni/installation-and-upgrade/container-deployment/uyuni/proxy-deployment-uyuni.html) or [Kubernetes](https://www.uyuni-project.org/uyuni-docs/en/uyuni/specialized-guides/kubernetes-guide/proxy-kubernetes-deployment.html).
